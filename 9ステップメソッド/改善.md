# 改善 ノウハウ

ジョブリスティング実践講座ステップ9の内容をベースに、フィードバックで肉付けしたもの。

最終更新：2026年4月5日

> **このファイルの使い方**
> 読んでわかった気になっても、一人で完結させることはできない設計になっている。
> 「事前に何をすればいいかを把握する」「フィードバック後に振り返って整理する」の2つに使う。
>
> **JLC**：各ステップで必ずフィードバックを受け、OKをもらってから次に進む。
> **JLCライト**：どのステップから始めても構わない。自分のペースで進める。

---

## 目的

- 配信開始後の広告を「継続的に改善する」ことで、費用対効果を最大化する
- 結果が出ない場合の原因を正しい順序で特定し、優先度の高いところから改善する

---

## 改善の全体像

```
配信開始直後
→ 除外キーワードの徹底（無駄クリックを止める）
→ 広告グループの整理（成果が出ないグループを停止）
→ 広告文の改善（クリック率を上げる → 品質スコアアップ → 単価ダウン）
→ サブキーワード対策（マッチしたページを追加作成）
→ 認知対策（社名検索への広告設定）
→ 他媒体への展開（リスティング以外に広げる）
```

---

## 進め方

### ステップ1：除外キーワードを毎日追加する（配信初期は特に重要）

**毎日の作業（慣れるまではルーティン化する）：**
1. 管理画面の「検索語句レポート」を開く（期間：今日・昨日）
2. 対象外の検索語句をリストアップ
3. 除外キーワードとして設定する

**除外の積み重ねで、徐々に無駄クリックが減り→作業量も減っていく。**  
最初は多い → 1日1回 → 週1回 → 月1回のペースに落としていく。

### ステップ2：入札単価を最適化する（週1ペース）

**管理画面で確認できる数字：**

| Google | Yahoo |
|---|---|
| 一番上に表示される単価 | 上位の単価 |
| 上部に表示される単価 | 1ページ目に表示される単価 |
| 1ページ目に表示される単価 | |

**判断基準：**
- 応募につながりやすいキーワード → 上位表示を狙って入札単価を上げる
- 単価が高すぎるキーワード → 低い方の表示位置に下げる
- 自動入札は使わない（月30万・月30コンバージョン以上の条件を満たさない場合は費用対効果が悪化する）

### ステップ3：広告文を改善する（クリック率を上げる）

**改善のサイクル：**
1. 2種類以上の広告文を作って同時配信する
2. クリック率の低い方を停止する
3. 新しい広告文を作成・配信する
4. 上記を繰り返す

**改善の効果：**
- クリック率が1%→3%（3倍）になった実績あり
- 年間5万回表示されていれば：500回クリック→1500回クリックになる
- 品質スコアが上がり→単価が下がる→同じ費用でさらに多く表示される（相乗効果）

**改善のポイント：**
- 広告見出し1の検索キーワードの入れ方・表現を変えてみる
- 見出し2でビッグプロミス・疑問形・証拠・ターゲット呼びかけを順番に試す
- 一貫してターゲットに向けた表現を保つ（ブレたらクリック率は下がる）

### ステップ4：リマーケティング広告を最適化する

**リマーケティング広告とは：** 一度採用LPにアクセスした人に、他サイト閲覧中に広告を表示して再誘導する。

**注意点：**
- 月間検索広告クリック数が1000回未満では、広告が表示されてもほとんど成果が出ない
- Google広告よりYahoo広告の方が少ないクリック数でも表示される
- 費用：Yahoo広告は月5000円前後が多い。Google広告は月1000クリック以上から

**やっておくべき最適化：**

1. **モバイルアプリを全除外（Google広告のみ）**  
   ゲームアプリ内での誤クリックを防ぐ。  
   「プレースメント除外」→App Store（84項目）・Google Play（56項目）を全選択

2. **Yahooは「レスポンシブ広告」を作成**（テキスト広告よりクリック率が高い）  
   レスポンシブ＋カルーセル広告を設定すると、表示回数が約3倍になる

3. **画像の選定**  
   クリックされる画像はターゲットが「自分のことだ」と感じるもの  
   （高齢者向けなら高齢のスタッフ写真が効果的）

### ステップ5：認知対策（社名検索への広告設定）

**仕組み：**
1. リスティング広告・リマーケティング広告で認知が広がる
2. 社名を覚えた求職者が「社名」で再検索する
3. 社名検索時に広告が表示される → コンバージョン率が最も高い

**社名検索の広告設定：**
- キーワード：社名・読み仮名・社名＋求人・社名＋採用
- マッチタイプ：完全一致
- 広告文：社名（キーワード）を含め、会社のベネフィット（条件面・職場環境）を中心に伝える

**注意：**
- 集客用ホームページも運営している場合は、社名のみの検索では集客用広告を表示させる、または採用ページへのリンクを貼る
- 求人用には「社名＋求人」「社名＋採用」のキーワードで対応する

**口コミ対策：**
- 社名検索すると右側にGoogleマイビジネスが表示され口コミも見られる
- 悪い口コミを書かれた場合：①Google削除申請 → ②新たな良い口コミで薄める（信頼できるスタッフや関係者にお願いする）

### ステップ6：サブキーワード対策（マッチしたページを追加作成）

**目的：** メインのキーワードに加えてサブキーワードでも応募が取れるようにする。

**手順：**
1. 広告配信の結果を分析して「サブキーワードからのコンバージョン率が低い」と判明
2. そのキーワードで検索している求職者に向けた新しいランディングページを作成
3. サブキーワードの広告はその新しいページに誘導する

**例（タクシー会社）：**
- メインページ：60代向けのメッセージ（60代住み込み・高齢住み込みで検索している人向け）
- 追加ページ：50代向けのメッセージ（50代はまだ定年になっていないため悩みが異なる）
- 結果：コンバージョン単価が36,845円 → 15,852円（2万円以上安くなった）

## 応募が入らない場合の対策（優先順位順）

大量のお金を使っても応募が1件も入らない場合、次の順番で改善を試みる。

| 優先順位 | 対策 | 理由 |
|---|---|---|
| **1. ターゲットを変える** | 未経験者OKにする・エリアを広げる・年齢層を広げる | 最も大きく結果が変わる |
| **2. ビッグプロミス（ヘッドライン）を変える** | 採用LPのコンセプトを変える | ページに最も大きく影響する |
| **3. デザインを見直す** | デザインのチェックリストを使って確認・修正 | ビッグプロミスがいいのに結果が出ない場合 |
| **4. コピー全体を見直す** | ロジカルチェーン・具体性・証拠を再確認 | 読み進めるほど「わからない」になっている可能性 |
| **5. 会社の環境を整える** | 給与・労働環境の改善 | ここまで来た場合はコピーではどうにもならない |

## 他媒体への展開（優先順位）

### オンライン媒体

**やらない方がいい媒体：**
- Facebook広告・YouTube広告 → 暇つぶし中の人に広告を出す → 仕事を探していないので応募につながりにくい

**優先順位：**
1. **ハローワーク（無料・効果実証済み）**  
   → 「ハローワークで見た → 社名検索 → 採用LPから応募」の流れが鉄板
2. **Google仕事検索のタグ設定**（クラウドソーシングに1万円程度で依頼可能）
3. **ジモティ**（無料・認知度そこそこ）
4. **SNS・メルマガ・ブログ（既に興味を持っている人向け）**  
   ただし「採用LPを作りました」と直接案内しても見てもらえない。  
   **橋渡しコンテンツ（スタッフインタビューの要約）をSNSで発信 → 採用LPへ誘導** が効果的
5. **インディードなどの有料求人サイト**（最終手段）

**SNS投稿のテンプレート（スタッフインタビュー活用）：**
```
① 前職を辞めた経緯・入社前の悩み
② 入社を決めた理由・きっかけ
③ 入社後の変化・得られた結果（ビフォーアフター）
④ こんな悩みを持っている人におすすめ → 採用LPURL
```

### オフライン媒体

**効果的な会社：** 多店舗経営（飲食店・美容室・スーパー等）  
→ 1種類の素材を全店舗に設置できる（レバレッジが効く）  
→ お客さんが働いている姿を直接見たり体験したりできる

**効果が出なかった媒体（実績より）：**
- タウンワーク（500万円使って採用0件の実績）
- 折り込みチラシ・フリーペーパー（効果が出にくい）

## 補足・深掘り

### やってはいけない4つのこと

1. **Google/Yahooサポートのアドバイスを鵜呑みにする**  
   「部分一致を使って」「自動入札に切り替えて」はほぼ毎回言われるが、従うと費用対効果が悪化する。設定方法や機能説明を聞くのは有効だが、運用方針は自分で判断する。

2. **最適化案を100%信じる**  
   管理画面の「最適化スコア」が低くても応募は入る。スコアを100%にするための変更（特に部分一致・自動入札）は受け入れない。

3. **マッチタイプを部分一致にする**  
   マッチングが崩れる。広告グループを細かく分けた作業が無意味になる。

4. **入札戦略を自動にする**  
   データ不足の状態（月30万円未満・月30コンバージョン未満）では費用対効果が2倍以上悪化した実績あり。
