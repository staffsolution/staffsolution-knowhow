# リスティング広告 ノウハウ

ジョブリスティング実践講座ステップ8の内容をベースに、フィードバックで肉付けしたもの。

最終更新：2026年4月7日

> **このファイルの使い方**
> 読んでわかった気になっても、一人で完結させることはできない設計になっている。
> 「事前に何をすればいいかを把握する」「フィードバック後に振り返って整理する」の2つに使う。
>
> **JLC**：各ステップで必ずフィードバックを受け、OKをもらってから次に進む。
> **JLCライト**：どのステップから始めても構わない。自分のペースで進める。

---

## 目的 {#purpose}

- 採用LPに「今まさに仕事を探している人」を的確に誘導する
- 広告費を最も無駄なく使える宣伝方法として先に習得する（SNS広告・フリーペーパーよりも先）

---

## 核心原則 {#core}

**「ユーザーが求めている情報を届ける」これだけ。**

- キーワード・広告文・ホームページの3つをできるだけ細かくマッチさせる
- マッチすればするほど品質スコアが上がり→クリック単価が下がり→費用対効果が上がる
- 欲張っていろんなキーワードで広告を出さない（最初は絞り込んでマッチさせる）

---

## 進め方 {#steps}

### ステップ1：キーワードのグループ分けをする {#step1}

**目的：** キーワードごとにマッチした広告文を設定するために分ける。

**グループ分けの考え方：**
- 同じような意図で検索しているキーワードをひとつのグループにまとめる
- グループごとに専用の広告文を作成する

**例（タクシー会社の場合）：**
- グループA：「高齢住み込み」関連キーワード → 高齢者向けの広告文
- グループB：「タクシー寮」関連キーワード → タクシー業界検討者向けの広告文
- グループC：「50代住み込み」関連キーワード → 50代向けの広告文

### ステップ2：マッチタイプを正しく設定する {#step2}

**使うべきマッチタイプ：フレーズ一致・完全一致のみ**

| マッチタイプ | 動作 | 判断 |
|---|---|---|
| **部分一致** | 関連語句・類義語まで幅広く表示 | **使わない**（マッチングが崩れる） |
| **フレーズ一致** | 設定キーワードが含まれる検索語句に表示 | 使う |
| **完全一致** | 設定キーワードに近い検索語句のみに表示 | 使う |

**部分一致がダメな理由：**
- 「美容室・求人」で設定すると「理髪店求人」「パート」まで広告が出る
- 広告グループを細かく分けた意味がなくなる
- Google/Yahooのサポートや管理画面の最適化案が「部分一致を使って」と勧めてくるが、これは広告費が増える設定のため従わない

### ステップ3：品質スコアを理解する {#step3}

**品質スコアとは：** GoogleがキーワードごとにつけるAd品質の評価（1〜10点）。

| 品質スコアが高いと | 品質スコアが低いと |
|---|---|
| 安い入札単価でも上位表示される | 高い入札単価でないと上位表示されない |
| 広告表示オプションが有効になる（目立つ） | 広告が表示されない場合もある |
| クリック単価が下がる | クリック単価が上がる |

**品質スコアは主にクリック率で決まる。**  
クリックされやすい広告を作るほど→品質スコアが上がる→単価が下がる→同じ費用でより多くの応募が獲得できる。

### ステップ4：広告文を作成する {#step4}

**広告文の構成（Googleレスポンシブ検索広告）：**

| 要素 | 重要度 | 入れる内容 |
|---|---|---|
| 広告見出し1 | ★★★最重要 | **検索キーワードを必ず含める**（スクロール中の指を止める） |
| 広告見出し2 | ★★★ | ビッグプロミス・疑問系・証拠・ターゲットへの呼びかけ |
| 広告見出し3 | ★ | 補足のベネフィット・特徴（表示されないこともある） |
| 表示URLのパス | ★ | 特徴・証拠を7文字以内で |
| 説明文1〜2 | ★★ | ベネフィット＋証拠をセットで伝える |

**広告文作成の鉄則：**
- 複数の広告文を作って戦わせる（最初から1本に決めない）
- 一貫してターゲットに向けてブレずに書く（シニア向けなら全部シニア向けで統一）
- クリック率が低い方を停止→新しい広告文で検証を繰り返す

**クリックされやすい広告文の表現パターン：**
- 検索キーワードを含める（絶対に外せない）
- ビッグプロミスを伝える（最大のベネフィット）
- ターゲットに直接呼びかける（「60代の方へ」「未経験OK」など）
- 疑問系で注意を引く（「タクシードライバーが60代から選ばれる理由は？」）
- 証拠を含める（「月収25万円実績あり」「定着率90%以上」）

### ステップ5：除外キーワードを設定する {#step5}

**除外キーワードは最初から徹底的に設定する。**  
放置すると無駄クリックが大量発生→広告費が一気に消える。

**事前に除外すべきキーワードの種類：**

| カテゴリー | 例 |
|---|---|
| 対象外エリア | 対象都道府県・市区町村以外の地名 |
| 異なる業種・職種 | タクシーを募集しているのにトラック・バス等 |
| 対象外の年齢層 | シニア・高齢・定年（若者を集めたい場合） |
| 対象外の性別 | 主婦・ママ・シングルマザー等 |
| 対象外の雇用形態 | アルバイト・パート・派遣（正社員のみ募集の場合） |
| 同業者が検索するキーワード | 美容室集客・美容室採用 |
| お客さんが検索するキーワード | 美容室予約・美容室営業時間 |

**除外キーワードの見つけ方：**
- Googleの検索ボックスで主要キーワードを入力→スペース→あいうえおを順番に入力して候補を確認
- 実際に広告を配信しながら「検索語句レポート」で見つかったものを追加していく

**重要：除外キーワードのマッチタイプは「部分一致のみ」で設定**  
（通常のマッチタイプと仕様が全く異なる。完全一致・フレーズ一致では除外できない）

### ステップ6：予算を設定して配信開始する {#step6}

**予算の目安：**
- テスト期間：15万〜20万円を最低限用意する（最初は検証用として使う覚悟で）
- Google：Yahoo：Yahoo DSP = 12万：2.5万：0.5万 を目安に

**配信前の最終チェック：**
- 除外キーワードリストがキャンペーン・広告グループに適用されているか（Yahooで忘れやすい）
- 除外エリアが設定されているか
- 配信を一時停止している状態から始めて、設定完了後に配信開始する

**配信直後は集中的にチェックする（最初の数時間〜数日が特に重要）：**
- 予算消化が早い → 即座に広告グループを整理する
- クリックが少ない → しばらく様子を見る

## 補足・深掘り {#supplement}

### 絶対にやってはいけないこと {#never-do}

| NG | 理由 |
|---|---|
| 部分一致でキーワードを設定する | 関連語句を広く拾いすぎてマッチングが崩れる |
| 自動入札に設定する | 月30万円・月30コンバージョン以上がない状態では費用対効果が悪化する（手動比較で2倍以上コスト増の実績あり） |
| Googleサポートのアドバイスを鵜呑みにする | 広告費を増やす方向のアドバイスが多い（「部分一致を使って」「自動入札に」が定番）。設定の仕方を聞くには使えるが、運用方針は自分で判断する |
| 最適化案を100%信じる | 管理画面の最適化スコアは参考程度。部分一致・自動入札への変更提案は特に無視でよい |

### 品質スコアが上がる仕組み（理解しておくと動き方が変わる） {#quality-score}

```
キーワードと広告文をマッチさせる
→ クリック率が上がる
→ 品質スコアが上がる
→ 上位表示に必要な入札単価が下がる
→ 同じ広告費でより多く表示・クリックされる
→ 同じ費用で獲得できる応募数が増える
```

少ないキーワードでも「徹底的にマッチさせる」ことが先。その後に広げていく。

## 【重要】検索パートナーの不正クリック問題と対策 {#click-fraud}

### 何が起きていたか

「管理画面のスコアは良い・コンバージョンも取れている・なのに応募が来ない」という状態が複数メンバーで発生した。

Googleサポートとの打ち合わせで原因が判明。**検索パートナーへの不正クリック**だった。

### 仕組み

管理画面の詳細を見ると、グーグル検索がほぼクリックされておらず、検索パートナーがメインになっていた。

検索パートナーの表示先をレポートで確認すると：

- **IndeeJapan**（世界規模で運営される海外サイト）
- **TopSeek**（英語サイト。日本の求職者はほぼ使わない）

これらのサイトから海外ユーザーが意図的にクリックしている可能性が非常に高い。確認した9アカウント全部でこの問題が発生していた。

### 正常な状態との比較

| 状態 | グーグル検索 | 検索パートナー |
|---|---|---|
| 正常 | 約5,400クリック | 約300クリック |
| 異常 | ほぼ0 | 大半 |

グーグル検索が圧倒的にメインになるのが正常。検索パートナーがメインになっていたら要注意。

### 対策手順

**1. 検索パートナーのチェックを外す**

管理画面 → キャンペーン → 設定 → ネットワーク → **「検索パートナーを含める」のチェックを外す** → 「ディスプレイネットワークを含める」も外す → 保存

これだけで問題は発生しなくなる。

**2. 入札戦略を手動（個別クリック単価）に戻す**

自動入札が不正クリックを学習してしまっている可能性があるため、いったん手動に戻す。ある程度コンバージョンが取れてきたら自動入札に移行してよい。

### チェックポイント

管理画面の数字が良くても、**「検索パートナー比率」を定期的に確認する習慣**をつける。比率がおかしければ即対処。

---

### デバイス別の入札単価調整 {#device-bid}

運用データが蓄積してきたら、デバイスごとのパフォーマンスを確認して調整する。

**判断の目安：**

| デバイス | 状況 | 対処 |
|---|---|---|
| タブレット | 半年で31件・コンバージョンゼロなど、極端に少ない | 入札単価調整費を100%引き下げ（実質停止） |
| パソコン | モバイルよりコンバージョンが少ない | 入札単価調整費を50%引き下げ（完全停止は避ける） |
| モバイル | 主力 | 調整しない |

**設定の場所：**
管理画面 → キャンペーン → 設定 → デバイス → 入札単価調整費

**考え方：**
予算を主力デバイス（モバイル）に集中させることで、同じ広告費でより多くの応募を獲得できる。
タブレットを完全停止しても影響はほぼない。パソコンは完全停止ではなく50%引き下げで様子を見る。
