---

## 目的

- 求人活動はゼロサムゲーム。競合と比較された時に真っ先に選ばれなければならない
- 脅威的な競合のチェックだけでなく、**比較されずに勝てるポジションを見つける**

**おおよその所要時間（感覚値）：**
- 未経験者：7日程度
- 経験者：3日程度

---

## 課題に取り組むコツ

- **ターゲットの立場に立って調べる**（ターゲットが何を一番重視しているか：ワークライフバランス・人間関係・やりがい・給料 など）
- リストアップするだけで終わりにせず、ちゃんと情報を見て整理する（ビッグプロミスも見つける・強いところをチェック・ランキングをつける）
- 「競合と比較された時に勝てるのか？」と自問自答する。どうしても無理だと思ったらターゲットの変更を検討する

## 進め方

### ステップ1：ゴールのイメージを掴む

競合リサーチのサンプルをチェックして、分析の方向性を理解する。

**競合の2種類：**

| 種類 | 定義 |
|---|---|
| **直接的競合** | 同じ業種・職種で求職者を募集している会社 |
| **間接的競合** | 異なる業種・職種で、自社がターゲットにしている求職者に向けて募集している会社 |

未経験者OKの募集は間接的競合が非常に多くなる（業種・職種を問わず同じ求職者を狙うため）。  
どちらの競合に広告を出すかによって、調べるべき会社が変わる。

**例（住み込み系募集の場合）：**
- 直接競合：同業他社
- 間接競合：トラックドライバー（住み込み）、期間工、新聞販売店、飛び込み系の職人

### ステップ2：競合情報を収集

**ネット検索：**
- ターゲットが検索するキーワードでGoogle検索
- リスティング広告の個別企業をチェック（リスティング欄の求人サイトは基本スルーでOK）

**ターゲットが使う求人サイトをAIで調べる：**
```
◯◯（ターゲット）が仕事を探す時によく使う求人サイトを教えてください
例：定年退職後のシニアが仕事を探す時によく使う求人サイトを教えてください
```

切り口別：
- 年齢・ライフスタイル（シニア・主婦）
- 雇用形態（正社員・パート）
- 業種・職種（警備・美容師・営業）

**求人ページで確認する3つのポイント：**

| ポイント | 確認内容 | 補足 |
|---|---|---|
| **① 雇用条件** | 給与・休日・福利厚生の相場 | 業界標準より明らかに悪いと選ばれにくい。住み込み系は特に手厚い競合が強い |
| **② ビッグプロミス** | ファーストビュー付近で何を一番訴求しているか | 競合が使っていない訴求軸が自社のポジショニングのヒント |
| **③ 応募方法** | 1ステップ（直接応募）か2ステップ（説明会・体験など経由）か | 競合が1ステップなら2ステップで差別化できる場合がある |

**競合の弱み・穴を必ずチェック（勝てるポジションを見つけるために）：**

| カテゴリ | 弱みの例 |
|---|---|
| **ターゲット層の抜け** | 誰でもOKにしていて絞り込んでいない。特定の層（シニア・未経験者・女性など）に刺さるメッセージがない |
| **福利厚生の弱さ** | 給与は高いが社会保険やボーナスがない。住み込み系なのに入社時サポートがない |
| **働き方の選択肢が少ない** | 時短勤務不可・副業NG・シフトの融通が利かない |
| **ビッグプロミスが弱い** | 「アットホームな職場」など抽象的で差別化になっていない |

### ステップ3：情報整理してフィードバックを受ける

- **マジックナンバー7を意識して7つ前後に整理**（人が一度に把握できる上限が7前後のため）
- 勝てるポジションを明確にする（集めた競合の情報を元に勝てるポジションを見つける）
- できたら、フィードバックを受ける

**差別化を言語化する手順：**
```
1. 「業界一般の問題」を3〜5つ書き出す
2. 自社がその問題をどう解決しているか書く
3. 「なぜ自社にはそれができるのか」の理由を書く
4. 競合他社が同じことをできない理由を書く（可能なら）
```

**「誰も言っていない訴求軸」と「自社が実現できること」を照合する。**  
競合が全員同じことを言っているなら、それを言わないことが差別化になる。

## 補足・深掘り

### 競合リサーチの3つの切り口

**1. 業界全体の「当たり前」を調べる**
```
（AI活用）○○（業種）の採用でよくある問題・不満を求職者目線で教えてください
（AI活用）○○業界の一般的な働き方・制度・待遇を教えてください
```
これを把握することで「業界の常識 vs 当社」という構図を作れる。

**2. 具体的な競合他社を調べる**
```
（AI活用）○○（業種）の仕事をしたい人が応募先を選ぶ基準を教えてください
↓出てきた基準を1つずつ深掘りする
```

**3. 「昔の自社」を競合として使う**
```
昔（業界の常識 or かつての自社）→ 今（当社）
```
**例（警備業界）：**
| 昔の業界常識 | 今の当社 |
|---|---|
| 急な呼び出しあり | なし（無理な受注をやめたから） |
| 連勤・長時間勤務 | 月24日・8時間勤務が基本 |
| 定年あり | 定年制度廃止 |
| 手取り不安定 | 月20万以上安定 |

### 繰り返し出るNGパターン

- 競合を調べずに自社の特徴だけ書く
- 「うちは良い会社です」と主張するが、他社との違いがわからない
- **リストアップして終わり**（強みのチェック・弱みの確認・ランキングをつけるところまでやる）
- ターゲットの立場から見ていない（求職者が何を一番重視しているかを軸に整理する）
- 業界の問題を知らないため、求職者の先入観（不安）に無自覚

### 業種別・競合の主な問題パターン

**美容・リラクゼーション系：**
- 新規客比率が高い → 予約が読めない → スタッフが疲弊
- 売上ノルマ・歩合 → 接客にプレッシャー
- 低単価・長時間労働

**建設・現場系：**
- 3K（きつい・汚い・危険）のイメージ
- 不安定な受注 → 残業・急な呼び出し
- 職人の上下関係・パワハラ体質

**医療・介護系：**
- 経験者優遇・未経験者が入りにくい
- 女性が多い・人間関係の複雑さ
- 体力的な消耗
