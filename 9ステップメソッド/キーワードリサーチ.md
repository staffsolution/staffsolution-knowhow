# キーワードリサーチ ノウハウ

ジョブリスティング実践講座ステップ1「キーワードリサーチの課題」の進め方をベースに、
フィードバック・事例・AIプロンプト型で肉付けしたもの。

最終更新：2026年4月5日

> **このファイルの使い方**
> 読んでわかった気になっても、一人で完結させることはできない設計になっている。
> 「事前に何をすればいいかを把握する」「フィードバック後に振り返って整理する」の2つに使う。
>

---

## 目的 {#purpose}

- ターゲットの方向性を全て洗い出す
- 「こんなはずではなかった」を潰す（ターゲットが変わると一からやり直しになるため）

**おおよその所要時間（感覚値）：**
- 未経験者：5日程度
- 経験者：2日程度

---

## 【重要前提】即戦力・経験者ターゲットは大手の戦略 {#important-premise}

経験者・資格保有者は母数が少ない上、どこも狙っていて競争が激しく、求職者側が選べる状態（モテる人がいろんな人からプロポーズされている状態）。必然的に条件合戦になる。大手でなければ戦いにくい。

| 業種 | 失敗ターゲット | 失敗の理由 | 成功ターゲット | 成功の理由 |
|---|---|---|---|---|
| **建設・建築** | 即戦力の経験者 | 母数少ない・競争激しい・広告費増やしても無駄 | 引きこもりから抜け出し自立したい若者（住み込み・寮で検索） | 全国配信で母数増・競合皆無 |
| **製造** | 即戦力の経験者 | 同上 | 派遣・バイトを転々している正社員になりたい若者 | 業種不問で正社員志望の母数が多い・競合皆無 |
| **警備** | 未経験OKの若い正社員 | 費用対効果が全く得られない | 定年退職後のシニア | 警備はシニアに認知されている・強い競合が少ない |
| **飲食店** | 未経験OKの正社員 | 業種イメージで避けられる | 学生アルバイト・主婦パート | 時短勤務が当たり前・気軽に応募できる・母数多い |
| **理・美容室** | 新卒の資格保有者 | 母数少ない・大手が多額の広告費 | 働きながら美容師になりたい若者 | 全国の資格なし層に届く→母数増・競合皆無 |

**どうしても経験者ターゲットを変えられない場合は、ハローワークでスカウトをコツコツやる方が現実的。**

## 進め方 {#steps}

### ステップ1：可能性のあるターゲットを全て洗い出す {#step1}

自分が「欲しい人材」にこだわらず、一度幅広いターゲット層をリストアップする。  
「求職者目線」で、どんな人がこの業界で仕事を探しているかを考える。

**ターゲット定義の4点セット：**

「30代女性・美容師経験あり」は属性の羅列であってターゲットではない。悩みと行動が決まって初めてターゲットとして機能する。

```
① 誰に（属性：年齢・職歴・家族構成など）
② 何を失いたくないか（回避したい不安）
③ 何を得たいか（願望）
④ 何と比較しているか（検討の競合軸）
```

**思考の流れ（時系列）で設計する：**

ターゲットがどうやって応募に至るかの時系列をたどる。この流れがわかると、LPに何をどの順番で書けばよいかが決まる。

例：医療事務未経験の中高年女性のケース
```
①老後不安、再就職を考え始める
②体力を考え、事務職に絞る
③一般事務の壁にぶつかる（経験者優遇が多い）
④医療事務にたどり着く（資格があれば有利では？）
⑤資格取得
⑥求人検索を始める（未経験可で検索）
⑦現実の壁にぶつかる（経験者優遇・若い人との競争）
⑧体力・人間関係の不安が出てくる
⑨応募前停止期に入る（本音は「不採用になるのが怖い」）
⑩職場選びの不安（ちゃんと教えてもらえる？残業は？）
```

**AIと壁打ちしながらターゲットを整理する：**
```
当社は◯◯業（業種）で◯◯（仕事内容）を行っています。
◯◯（目的）なので、◯◯（欲しい人材像）の応募を年内に◯名集め、
その中から◯名採用したいです。
◯◯（エリア）にリスティング広告で集めるので、
◯◯（願望・不安・疑問点など）と感じています。
この考えはあっていますか？また、他にどのようなターゲット設定が考えられますか？
```

### ステップ2：フィードバックを受けてメインターゲットの当たりをつける {#step2}

メインターゲットやサブターゲットの当たりが付いたら、フィードバックを受ける。  
**最初からターゲットを明確に決めなくてもOK。** その後のステップで絞り込んでいく。

**メイン・サブの切り分け方：**

ターゲットが複数混ざると訴求がぼやける。

```
メインターゲット（1人）：全力で刺しにいく
サブターゲット（2人まで）：後で対応する旨を明記し、情報の混線を防ぐ
```

切り分けの基準：
- 今すぐ応募してほしい人は誰か
- 数が最も多い層はどこか
- 自社が最も価値を発揮できる層はどこか

**優先順位の決め方（複数候補がある場合）：**

例：貸しスペースの場合

| 時間帯 | ターゲット | 優先度 |
|---|---|---|
| 平日昼 | 法人（外部会議・営業拠点） | ★★★ |
| 平日夜 | スクール・勉強会 | ★★ |
| 土日 | セミナー・イベント・採用説明会 | ★★ |

「常に埋まっている状態」を目標とするとき、時間帯ごとにターゲットを分けるとLPも2枚に分けやすくなる。

**この段階はあくまで仮設定：**

手順：
1. 憶測でいいのでターゲットを仮に出す
2. スタッフインタビューで答え合わせする
3. ズレていたら全て訂正する

仮設定なしでインタビューすると、得られた情報が活かしきれない。

**ターゲット設定のポイント：**
- 未経験者と経験者の2つのターゲットがいるなら、**メインは未経験者にする**
- 経験者をメインにしたメッセージは未経験者が反応しないが、未経験者向けに作ったメッセージは経験者も反応する
- 一人に絞り込むと多くの人が反応する：一人の行動パターンを映像でイメージできると、キーワードもメッセージも精度が上がる

### ステップ3：候補ターゲットの検索キーワードを洗い出す {#step3}

AIと壁打ちしながら情報収集し、マインドマップに整理する。  
**キーワードは全て洗い出さなくてOK**（グループ名がわかれば十分）。

**AIプロンプト型：**
```
（主要キーワードを探す）
◯◯（ターゲット）は、どんな検索キーワードで仕事探しをしますか？
例：働きながら美容師免許を取得したい人は、どんな検索キーワードで仕事探しをしますか？

（関連語句・類義語を探す）
◯◯（資格名・職種名・業種名）は別の名称でなんと言われていますか？
```

**キーワードの種類（グループ分け）：**

| 分類 | 例 | 特徴 |
|---|---|---|
| **〜したいキーワード（メイン）** | 「美容師 求人」「タクシー 採用」 | 応募につながりやすい。競合も多い。最優先で押さえる |
| **〜を知りたいキーワード** | 「美容師 年収」「介護士 つらい」 | 情報収集目的。応募につながりにくい。エリアが狭い場合のみ検討 |
| **〜に行きたいキーワード（自社指名系）** | 「◯◯会社 採用」「インディード」 | 認知済みの人向け。競合他社名+採用も積極的に狙う |
| **業種直接系** | 「鳶 求人」「警備 正社員」 | 検索意図明確・競合多い |
| **関連職種系** | 「玉掛け 求人」「フォークリフト 正社員」 | 類似職種からの流入・比較的穴場 |
| **地域系** | 「新潟 正社員」「水戸 美容師」 | エリア絞り込み |
| **ライフスタイル系（メイン向け）** | 「住み込み 正社員」「寮 求人」「20代 正社員」 | ターゲットの状況に直結するキーワード |

**地域名キーワードの注意点：**
- 地域名はすべてのキーワードに含めなくてよい（配信地域の設定で対応可能）
- **【注意】集客キーワードになっていないかチェック**：「水戸 美容室」「水戸 リフォーム」などはお客様向けキーワードになるため、採用広告では無駄クリックになる
- 地域名と組み合わせてOKなのは：求人関連・雇用形態・資格名（例：水戸 求人、水戸 正社員、水戸 美容師）

**年齢・性別の絞り込みはキーワードでなくても可能：**  
Google広告では年齢・性別・世帯収入・子供の有無での絞り込み配信ができる。

**マインドマップに当てはめたらフィードバックを受ける。**

### ステップ4：キーワードプランナーで数値感を掴む {#step4}

数値化せず、**感覚値でOK**。  
メインターゲットはサブターゲットよりどのくらい検索ボリュームが多いか、単価の差はどのくらいか、程度を把握する。

**キーワードプランナーの正しい使い方：**
- **使う機能：** 「新しいキーワードを見つける」のみ。「検索のボリュームと予測のデータを確認する」は当てにならない
- **調べ方：** 掛け合わせキーワードで入力する（単体のみはNG）
- **期間設定：** 直近だけでなく1年分の推移を確認して季節変動を把握する
- **Googleのみでよい：** GoogleはYahooの約3〜4倍のアクセスのため、キーワードプランナーのみで十分

**一度報告して次のステップに移る。**

## 次のステップに進むためのチェックリスト {#checklist}

- ✅ メインターゲットとサブターゲットが明確になっているか？
- ✅ クリック単価と検索ボリュームを比較し、適切なターゲットを選んだか？
- ✅ 採用目標に対して、十分な応募数を確保できるイメージは持てているか？

## 補足・深掘り {#supplement}

### 繰り返し出るNGパターン {#ng}

- キーワードプランナーを使わず、思いつきで並べる
- 検索ボリュームを確認せずに優先順位を決める
- ターゲットを決める前にキーワードを探し始める
- 業種直接系しか考えず、関連職種・ライフスタイル軸を見落とす
- 地域キーワードを集客キーワードと混同して無駄クリックを生む

### フィードバックから抽出した業種別キーワード実例 {#keyword-examples}

**重量鳶・建設系（未経験者向け）：**
- メイン：「住み込み 正社員」「寮 求人」「20代 正社員」
- 業種関連（サブ）：「鳶 求人」「建設 転職」「玉掛け 求人」「足場 求人」
- キャリア軸：「手に職 求人」「資格取得支援 求人」
- 安定軸：「安定 求人」「長く働ける 求人」

**警備（シニア向け）：**
- メイン：「シニア 住み込み」「60代 正社員」「定年後 仕事」
- 業種関連（サブ）：「警備 求人」「警備 正社員」

### お宝キーワードとは {#treasure-keywords}

「〜したいキーワード」の中に、競合が見落としている掛け合わせキーワードが存在する。  
検索ボリュームが多くてクリック単価が安いキーワードを「お宝キーワード」と呼ぶ。

**見つけ方：** キーワードプランナーで業種・職種・地域・雇用形態・条件などの掛け合わせを調べる。
