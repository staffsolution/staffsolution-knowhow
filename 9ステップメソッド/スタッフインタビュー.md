## 目的 {#purpose}

- 求職者の深層心理（BDF）を把握すること
- 企業の強みや魅力を、求職者の言葉で表現すること
- 採用活動全体の方向性を定めるために必要な情報を収集すること
- ポジショニングを確定する

**広告は常に相手ごと。** 求職者がどんな結果・変化を求めているかを知らないと、何を伝えれば反応するかがわからない。  
スタッフに徹底的にインタビューすることで、**「求職者の頭の中の会話」**に忍び込めるようになる。

**おおよその所要時間（感覚値）：**
- 社員3〜5人を選んで動画撮影、1人1.5〜2時間
- 文字起こし・要約はAIにやってもらえば早い
- 合計で8時間程度

## スタッフインタビューの注意点 {#cautions}

### スタッフ選択時

- **レアケースに注意**：ターゲットにマッチしていないけど良い人だからインタビューしたい、はNG。ターゲットにピッタリマッチする人材を選ぶ
- 事前に動画掲載の許可を貰えるように努力する（詳細は下の「動画撮影の同意を得るコツ」参照）

### 動画撮影の同意を得るコツ

基本、ネットに自分の情報が掲載されるのは嫌がられる。それでも動画のある・なしでは信用性に大きな差が出るため、**できるだけ食い下がって同意を得る**。

**説得の流れ（伝え方の例）：**
```
・動画のある無しでは信用性に大きな差が出る
・良い人材を採用できなければ、会社に未来はない
・給料を増やすことはもちろん、維持することすら難しくなる可能性もある
・今の世の中、人口が減少しているため、普通に求人を出すだけではなかなか応募してもらえない
・どんな人が働いているのか、スタッフの生の声を動画で見てもらうことで、
　働きたいと思ってもらえる可能性が高まる
・あなたのインタビューがきっかけで、会社にマッチする新しい仲間が増えれば、
　職場環境はより良くなっていくと思いませんか？
・YouTubeに掲載したとしても、ほとんど見られないのが実態
・名前を伏せることも可能だし、使って欲しくないところは編集できる
・問題が起きることはないので、安心してほしい
```

嫌がられてもインタビュー自体は必ず実施する（動画なしでもネタとして使える）。  
辞めた後も使えるように事前に約束しておくとベター。

### インタビュー時の事前準備

- 背景に変なものが映らないように
- 顔の位置が適切かどうか確認
- 窓が開いていないかチェック（音が入るため）
- 質問項目を用意する

### インタビューできる人がいない場合のAI代替

**手順：**

**ステップ1：ペルソナを決める**
```
子育て終了した主婦が仕事をしたいと思って探しているとします。
よくいる主婦のペルソナを出してください。
```

**ステップ2：ペルソナを選んでインタビューを依頼する**
```
◯◯さん（ペルソナ名）が美容室&リラクゼーションの会社で働くようになり、
楽しく過ごしているとします。
そのような想定でスタッフインタビューを進めたいと思いますが、答えられますか？
```

**ステップ3：質問項目に沿って聞いていく**  
通常のインタビュー項目と同様に、前職・応募・入社後の流れで深掘りする。

## 進め方 {#steps}

### ステップ1：ターゲットに該当するスタッフを3〜5人選ぶ {#step1}

- あくまでも設定しているターゲット層に近いスタッフを選ぶ
- 事前に動画掲載の許可を得ておく
- 念の為、選んだスタッフが問題ないかどうか報告してフィードバックを受ける

### ステップ2：スタッフインタビューの質問項目をサンプルを元に決める {#step2}

**引き出すべき3種類の情報（BDF）：**

| 記号 | 意味 | 内容 |
|---|---|---|
| **B** | Belief（ビリーフ） | 求職者が思い込んでいること・当たり前だと思っていること・業界に対するイメージ |
| **D** | Desire（デザイア） | 求職者の欲求・願望・避けたいこと |
| **F** | Feeling（フィーリング） | 業界・仕事・会社に対してどう感じているか（ポジティブ・ネガティブ両方） |

**Beliefが最も重要で最も引き出しにくい。**  
当たり前のことは本人が意識していないため答えにくい。具体的な例え話で「思い出すきっかけ」を与えると引き出しやすくなる。

**インタビューの時系列（4つのカテゴリー）：**
```
1. 前職を辞めた時のこと
   → 何に不満があったか、何が辛かったか、なぜ動いたか

2. この会社に応募した時のこと
   → 他社との比較、求人・ページを見た時の印象、応募した決め手

3. 入社後の変化
   → 入社前と後で何が変わったか、具体的なシーン、周りからの反応

4. 現在の仕事内容・職場環境
   → 日常の仕事の様子（コピー作成上の比重は低いが現実感を出すために必要）
```

**最も重要なのは1〜3。**

**入社後の変化までの共通質問に加えて、独自の特徴に対してのイメージやベネフィットを聞く：**
- 研修について
- 1日の流れ
- 社風について
- 福利厚生について
- 休みや時間についてなど（自社リサーチのネタがベースになる）

**質問項目ができたら、報告してフィードバックを受ける。**

### ステップ3：動画撮影 {#step3}

- 深掘りの質問を意識してインタビュー
- 業界や会社のことを全く知らない体でインタビューする（専門用語や抽象的な言葉が出てきたら、絶対にスルーしてはいけない）

**深掘りの技術（縦と横の2方向で掘る）：**

**縦に深掘りする（1つのテーマを掘り下げる）：**
- 「例えば、具体的にはどんなことですか？」（事例・エピソードを引き出す）
- 「なぜそう感じたんですか？」（理由を引き出す）
- 「過去の職場や他の何かと比べてどうでしたか？」（比較で差を際立てる）

**横に広げる（別の切り口を探す）：**
- 「他には何かありますか？」

**インタビュー開始前に告知しておく（突っ込んで聞くと詰められていると感じる人もいるため）：**
```
「本音の気持ちを知るために突っ込んで質問させていただきます。
もしかしたら失礼だと感じることもあるかもしれませんが、
必要なプロセスですので、あらかじめご了承いただければ幸いです。」
```

### ステップ4：音声を文字起こし {#step4}

- 撮影した動画はSlackで文字起こし（音声のみの場合も同様）
- 文字起こししたテキストはAIで綺麗にする

```
（AI活用）こちらの文字起こしを綺麗にして、インタビュアーと答える人がわかるようにしてください。
--
コピペで文字起こしの情報を入れる
```

**マインドマップで整理する：**
- 趣味や家族構成などの自己紹介は要約してOK
- **BDF（Belief・Desire・Feeling）に該当するところは要約せず、話し言葉をそのまま入れる**（コピーライティングで「求職者の言葉」として使うため、意味が変わる要約はNG）
- 1人目のマインドマップができた時点でフィードバックを受ける

### ステップ5：全スタッフの情報をマインドマップで整理する {#step5}

- 全スタッフの情報は**質問項目ごとに一つのファイルにまとめておく**（次のコピーライティングステップで使いやすくなる）
- 綺麗に整理できたら、報告してフィードバックを受ける

- ✅ わかりやすく情報整理できているか？

## 補足・深掘り {#supplement}

### 繰り返し出るNGパターン {#ng}

- ペルソナを決める前にインタビュー相手を選ぶ
- 入社後の感想だけ聞いて、「なぜ選んだか」を省略する
- 質問を羅列するだけで深掘りしない
- 「良かったです」「安心しました」で終わらせる（具体的なシーン・数値を引き出さない）
- ホームページや求人を見たときの印象を聞かない

### インタビューで使える情報と使いにくい情報を判断する {#usability}

全スタッフのインタビューを同等に扱わない。情報の活用判断の基準は「大多数のターゲットに当てはまるか」。

**繰り返し語られるテーマが一番強い欲求のサイン**

複数のスタッフが同じようなことを繰り返し語っているテーマは、ターゲット全体に共通する一番強い欲求・感情を表していることが多い。そのテーマを訴求の核に使う。

**独特な考えを持つスタッフはターゲット代表にしない**

自分の中の前提・思い込みが強く、独特な考え方をするスタッフの発言は、「大多数に当てはまるか」で精査する。

判断基準：
- 「大多数のターゲットが同じように感じるか」→ 使える
- 「この人特有の考え方で、他の人は違うかもしれない」→ 限定的に使う or 使わない

思い込みが強い方のインタビューでは、発言の前提を「それは事実ですか、それともそう思っていたということですか」と確認しながら聞く。事実と思い込みが混在していることがある。

**一度辞めて戻ってきた経緯を聞く**

一度退職した後に同じ職場に戻ったスタッフがいる場合、「なぜ戻ってきたのか」を聞く。他の職場を経験した上でここを選び直した理由に、このお店の隠れた強みが出てくることが多い。

---

### 「入社前の葛藤と決断」を引き出す（最重要） {#turning-point}

インタビューの価値は「入社前の葛藤と決断」を引き出せるかどうかで決まる。  
入社後の感想だけ聞いても、応募者の背中を押す言葉にはならない。  
「なぜ動いたか」「何が決め手だったか」の部分が一番使える素材。

**深掘り型の質問例：**
```
・前の職場、何年いましたか？
・変えようと思ったとき、何が一番引っかかりましたか？
・家族や誰かに相談しましたか？その人はなんと言っていましたか？
・他の会社も見ていましたか？最終的になぜここにしたんですか？
・ホームページを最初に見たとき、どんなことが気になりましたか？
・「これは自分に合わないかも」と思った部分はありましたか？
```

### 社長・幹部インタビューの意味 {#executive}

社長や幹部からのメッセージがあることで、求職者が「この会社の雰囲気・上司との関係性」を事前にイメージできる。  
人間関係を重視する求職者への訴求に特に有効。ミスマッチ採用を防ぐ効果もある。

### お客様インタビューへの応用 {#customer}

スタッフインタビューと基本構造は同じ。「来店前の葛藤と決断」が最重要な点も同じ。

- フェーズ1：年齢・家族構成・仕事・趣味・普段の生活
- フェーズ2：これまで使っていたお店の不満・変えようと思ったきっかけ・サービスに対して持っていたイメージ・他のお店と比較検討したか
- フェーズ3：来店のきっかけと決断・HPを見たときの印象・最終的に決めたポイント・来店後の変化
