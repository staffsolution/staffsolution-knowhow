## 目的

- 求人で成果を出せるかどうかはこのポジショニング次第（成否の8割を決める）
- No.1になれるカテゴリーを自分で作る
- 競合他社が攻めていないところ・会社の強み・求職者が求めていることが合致するところを見つける
- 絞り込むことでコアとなるアイデアを見つけ出す
- 出したアイデアでいけるかどうかは次のステップ（スタッフインタビュー）で確認するため、仮のアイデアで問題なし

**おおよその所要時間（感覚値）：**
- 未経験者：2〜3日程度
- 経験者：1日程度

## 進め方

### ステップ1：サンプルのマインドマップを参考にしてゴールのイメージを掴む

- ポジショニングのサンプルの解説を確認する

**ポジショニングの3つの問い（良いポジショニングは3つが重なる場所にある）：**
1. 求職者が強く求めているもの（ターゲットの欲求・悩み）
2. 競合が提供していないもの（競合のビッグプロミスで使われていない訴求軸）
3. 自社が本当に提供できるもの（根拠のある強み）

### ステップ2：自社リサーチの情報を綺麗に整理しておく

情報整理ができていなかったり、不必要な情報が入っているとAIがうまく機能しないため、  
自社リサーチの情報を整理した状態でAIに渡す。

### ステップ3：AIと対話しながら情報を出していく

**4つの質問をAIに順番に投げる（AI任せにせず、自分の頭で考えた上でヒントを求める）：**

**① ベネフィットを5つ出す**
```
この自社リサーチの内容から求職者が入社後に得られるベネフィットを5つ出してもらえますか？
--
（自社リサーチの情報をコピペ）
```
→ 見出しが抽象的なら「見出しをもっと具体的にしてください」と追加指示。難しい表現は「○○をわかりやすくして」。

**② ターゲットが本当に求めているものを聞く**
```
◯◯◯（ターゲット）が本当に求めているもの・問題は何ですか？
```
→ 情報量が多い場合は「1つか2つに絞り込んで」と指示。

**③ なぜそれが欲しいかを深掘る**
```
なぜ、それがほしいのですか？入社して求職者はどうなりたいのですか？
```
→ 出てきた情報から使えそうなところを抜き出す。

**④ 条件以外の強みを絞り込む**
```
条件以外で求職者が本当に求めていることを満たす自社のベネフィットは何ですか？
```
→ 「1つか2つに絞り込んで」と指示。

**⑤ ターゲットと強みのどちらで絞るか決める**
```
求職者で絞るのでしょうか？会社のベネフィットで絞るのでしょうか？それとも両方でしょうか？
```
→ AIの答えを参考にしながら、自分のアイデアを出す。

**まとまったら、フィードバックを受ける。**

### ステップ4：情報整理して綺麗にまとめていく

- AIの情報は重複しているところも多い
- 表現がイマイチなケースも多い
- 情報を削除したり表現を変えたりして綺麗に整理していく
- うまくまとまったと思ったら、フィードバックを受ける

## 補足・深掘り

### 既存のカテゴリーから抜け出す

同じカテゴリーの中で戦うと、最終的に条件（給与・休日）の勝負になる。  
**「比べられない土俵を作ること」** が勝てる方法。

```
悪い例：「求人媒体で成果を出せます」→ 求人広告会社というカテゴリーの中で埋もれる
良い例：「求人媒体を一切使わない採用の仕組みを提供します」
　　　　→「求人媒体 vs 求人媒体を使わない方法」という新しい対比軸が生まれる
```

ターゲットが「比較検討している業態・カテゴリー」を把握し、そこから自社を切り離す言い方を探す。

### 書き方の鉄則：3点セットで書く

```
① 何をしているか（特徴・制度・仕組み）
② なぜそうしているのか（理由・背景）
③ それによってどうなるか（求職者へのベネフィット）
```
どれか1つが抜けると「だから何？」になる。

### 因果関係でつなげる

```
A（特徴）だから
　↓
B（中間の結果）になる
　↓
C（求職者にとっての変化）
```

**良い例（完全予約制の場合）：**
```
完全予約制で前月に全て埋まる
　↓来店数を予測できる
　↓スタッフが余裕を持って準備できる
　↓1人1人の施術に集中できる
```

**やりがちなNG：**  
「完全予約制で安心して働けます」← なぜ安心かが飛んでいる

### 読者の「でも、どうせ…」を先回りする

```
「月20万以上稼げる」
　↓（読者の頭の中）「でも急な呼び出しとかあるんじゃないの？」
　↓（先回りして処理）「実は当社も昔は同じでした。無理な受注をやめたことで
　　急な呼び出しも連勤もなくなりました。」
　↓（強みを再提示）「だからシニアでも月24日・手取り20万以上を実現できる」
```

### 繰り返し出るNGパターン

- 「何をしているか」だけで「なぜ」がない
- 比較なしに強みだけ並べる
- 抽象語（安心・成長・やりがい）で止まる
- 重複を放置
- 「全員向け」に書く（結果誰にも刺さらない）

### 抽象語は必ず具体行動に置き換える

| 抽象 | 具体 |
|---|---|
| 安心して働ける | 月○日勤務・残業なし・急な呼び出しなし |
| 成長できる | 入社○ヶ月で○○の資格取得・先輩からのフィードバック週○回 |
| アットホームな職場 | 年下が先輩でも本音で話し合える・社員同士で旅行に行く |
| 働きやすい環境 | 完全予約制・1日2〜3名の施術・5.5時間固定シフト |
