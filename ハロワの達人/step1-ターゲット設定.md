# Step1 ターゲット設定

[← 目次に戻る](./index.md)

---

## なぜ重要か

ターゲット設定が曖昧だと、この後のすべてのステップに影響する。

- 競合リサーチで見るべきポイントが分からない
- 自社リサーチで強みが見つけられない
- 求人票が誰にも響かない

逆に明確にできれば、競合リサーチの視点が定まり、求人票がターゲットに刺さるようになる。

---

## よくある失敗：即戦力・経験者をターゲットにする

- そもそも母数が少なく、競争が激しい
- 資金力のある大手が有利で中小企業が選ばれにくい
- ハローワークの相談員が未経験求人の方が紹介しやすい（対応の幅が広がる）

**未経験をターゲットにして出した結果、経験者が応募してくるケースも多い。**

---

## 進め方（3ステップ）

### Step1-1：可能性のあるターゲットを10個書き出す

「欲しい人」ではなく**「来てくれそうな人」**の視点で考える。

**考えるヒント：**
- 今活躍しているスタッフはどんな経歴で入社したか
- 地域・条件から逆算する
  - 未経験OKなら → 異業種転職者、未経験者
  - 短時間勤務なら → 主婦、シニア
  - 土日休みなら → 子育て世代
  - 夜勤ありなら → 若者、フリーター
  - 住宅街立地なら → 近隣の主婦

**意外と合う人の例（介護の場合）：**
- 飲食接客経験者 → コミュニケーション力が高い
- 子育てが落ち着いた主婦 → 働く意欲が高く続けやすい
- 保育経験者 → 声かけや環境づくりのスキルが応用できる

**ターゲット候補の例：**
未経験者・子育てが落ち着いた主婦・シニア・フリーター・定年後も働きたい人・手に職をつけたい人・異業種転職者・安定を求めている人 など

困ったらAIに聞く：
```
○○業で○○市にハローワーク求人を出します。
どんな人が応募してくれそうですか？10個教えてください。
```

---

### Step1-2：現実的なターゲットを3〜5つに絞る

絞る視点：
- 来てくれる可能性が高いか
- 母数が確保できるか
- 自社で育成できるか

AIに相談する場合：
```
先ほど挙げた10個の中で、現実的に来てくれる可能性が高い順に並べてください。
```

**メインターゲット1つ＋サブターゲット複数**の形に整理する。ハローワークは無料なので複数パターンのテストができる。

---

### Step1-3：1人の具体的な人物像をイメージする（ペルソナ設定）

基本情報を仮設定する（名前・年齢・性別・家族構成・現在の状況）。

その人の**段階別の悩み**を掘り下げる：
1. 仕事探しを始めた時の悩み
2. この業界・職種を検討する時の悩み
3. 会社を選ぶ時の悩み

AIに聞く例：
```
ターゲット「○○」はどんな悩みがありますか？
仕事探しの段階で悩むことは？
この業界を検討する時の悩みは？
```

---

## まとめフォーマット

```
【基本情報】業種・職種・勤務地・雇用形態

【ターゲット10候補】
1. ○○
2. ○○
...

【絞り込み（3〜5つ）】
メイン：○○
サブ：○○、○○

【ペルソナ（1人）】
名前：田中さん（仮）
年齢：42歳、家族構成：子ども2人（中高生）
現在の状況：子育てが落ち着いてパートを探し始めた

悩み・願望：
- ブランクがあっても安心して働きたい
- 未経験でもちゃんと教えてほしい
- 家庭と両立したい・急な休みも取れる職場

段階別の悩み：
- 仕事探しの時：専業主婦が長くて何をすればいいか分からない
- 業界検討時：資格も経験もないけど本当に大丈夫か
- 会社選択時：ちゃんと教えてもらえるか、若い人の中で浮かないか
```

最初は仮決めでOK。この後のリサーチを進めながら変更していく。
