# Step2 競合リサーチ

## 目的

**自社が勝てるポイントを見つけること。** 真似をするためではない。

競合を知ることで：
- 強い競合がどこを攻めているか分かる
- 競合が攻めていない空白地帯が見える
- 自社がどこで差別化できるかが見えてくる

---

## よくある失敗：強い競合の表現を真似する

「未経験でも安心」「丁寧に指導します」「アットホームな職場」——どこでも同じことが書いてあって差別化にならない。結局、条件（給料・休日）で選ばれることになる。

---

## 競合の種類

**直接競合：** 同業種・同職種の求人
- 介護職募集 → 他の介護施設・訪問介護・デイサービス

**間接競合：** 違う業界でも同じターゲットを狙う求人
- 主婦向けパート介護職 → スーパーのレジ・学校給食・清掃パート
- 若者向け正社員 → 飲食店・コンビニの正社員

**ターゲットが変われば競合も変わる。** 同じ介護職でもシニア向けと若者向けでは競合が全く異なる。

---

## 進め方

### 1. ハローワークインターネットサービスで検索する

[ハローワークインターネットサービス](https://www.hellowork.mhlw.go.jp/) → 「求人情報検索」

**直接競合を探す場合：**
- 地域：自社の通勤圏内
- 雇用形態：自社と同じ
- 職種：自社と同じ職種

**間接競合を探す場合：**
- 地域：同じ
- 職種：指定しない（ターゲットが比較しそうな別職種）
- 詳細条件：時給・休日・勤務時間などを自社に近い条件で絞る

強そうな求人を5社程度ピックアップする。

**注意：** 求人票には公開期限がある。良い求人を見つけたらPDFをダウンロードして保存しておく。

### 2. 各求人の情報を集める

各求人票から以下を確認：
- 雇用条件で目立つもの
- ビッグプロミス（最も魅力的な約束）
- ターゲットにとって目を引く表現
- 詳細ページ（会社URL・事業所からのメッセージ・福利厚生）

記録する情報：求人番号・詳細ページURL・求人票PDF

### 3. フォーマットに整理する（5社分）

| 項目 | 内容 |
|---|---|
| 会社名 | |
| 求人番号 | |
| 詳細ページURL | |
| 競合とした理由 | 例：主婦パートで条件が近いため |
| 強い雇用条件 | 例：時給1,200円・週3日OK |
| ビッグプロミス | 例：未経験から介護福祉士を目指せます |
| 具体的な数字 | 例：研修は最初の1ヶ月マンツーマン |
| 強いと思った点 | 例：急な休み対応と固定シフトで主婦層に強そう |
| 自社が勝てそうな点 | 例：給料では勝てないが教育の具体性で差別化できる |

### 4. 強い順に並べて比較する

**強い競合の共通点：**
- 具体的な数字がある（研修期間・シフト例など）
- ターゲットの不安に先回りして答えている
- 応募ハードルが低い（LINE応募・出張面接など）

**弱い競合の特徴：**
- 抽象語だけ（「丁寧に指導します」「アットホーム」）
- 根拠がない
- ターゲットの悩みに触れていない

---

## 自社の勝ち筋を考える

例：
- 家庭優先で働ける主婦向けに急な休み対応と短時間固定シフトを約束する
- 未経験でも1ヶ月で一人立ちできるように研修の道筋を具体的に見せる
- 給与では勝てないが人間関係の安心と教育の具体性で選ばれる

---

## AIを使った競合分析

求人票をAIに貼り付けて分析させるボットも活用可能（イベントで共有）。どこが強みか・ターゲットの引力がどれくらいかを抽出してくれる。

---

## 競合リサーチ記入フォーマット

5社分コピーして使ってください。

```
【会社名1】
求人番号：
詳細ページURL：
PDF保存URL（任意）：

なぜ競合としたか（ターゲットが比較しそうな理由・どのターゲットを想定しているか）：

雇用条件 目立つ点（最大3つ・数字や具体を入れる）：

福利厚生 目立つ点（ターゲットに刺さるもの最大3つ・意味づけする）：

ビッグプロミス一言（入社したらどうなるかをターゲット向けに一文）：

根拠・具体情報・数字・事例（ビッグプロミスを信じられる根拠）：

応募方法・応募のしやすさ（応募の手間と面接までのハードルを一言評価）：

強いと思った点（この求人が選ばれる理由を一文）：

自社が勝てるとしたら（真正面から勝負しない・勝てる角度を一つ決める）：
```

**記入後は強い順に並べて比較する。** 共通点と違いを把握し、自社の勝ち筋を考える。

**リンクの保存方法（XMindやノート使用時）：**
- Mac：Command + K
- Windows：Ctrl + K
- または「挿入」→「リンク」から挿入
