# ジョブリスティング 実践ノウハウ

ジョブリスティングを内製化するための実践ノウハウ集です。9ステップメソッドに沿って正しく取り組むことで、求人と集客を自社で回せるようになります。フィードバックや事例をもとに随時更新します。

---

## このノウハウ集の使い方

各ページは「読んでわかった気になる」ためではなく、**AIと一緒に作業するための指示書**として使うと効果が出ます。

### ステップ1：ノウハウを読んでからAIに作業を依頼する

各ページの「進め方」と「AIプロンプト例」をそのままChatGPTやGeminiにコピーして貼る。
「何を聞けばいいかわからない」という状態がなくなり、作業が一気に進む。

```
（使い方の例）
キーワードリサーチのページを開く
 → 「ステップ1」のAIプロンプトをコピー
 → ChatGPTに貼って、自社の情報を追記して送信
 → 出てきた候補をもとにフィードバックを受ける
```

### ステップ2：ノウハウをAIに読み込ませて壁打ちする

ノウハウのページ全文をコピーしてAIに貼り、こう伝える。

```
このノウハウに基づいて、
私の会社（◯◯業・◯◯エリア）の場合を一緒に考えてください。
```

AIが「このノウハウを知っている前提の壁打ち相手」として機能する。
自分の理解があいまいな部分も、AIとのやり取りの中で整理できる。

### ステップ3：AIの出力をチェックリストで確認する

AIが出した内容を、各ページ末尾の「次のステップに進むためのチェックリスト」と照合する。
自己採点として使える。「何が足りないか」が明確になる。

---

> **JLC**：チェックリストでOKになったらフィードバックを受ける。OKが出てから次に進む。
> **JLCライト**：チェックリストを自己確認しながら自分のペースで進める。
